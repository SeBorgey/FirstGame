# FirstGame

Простое обучающее Android-приложение для детей, обучающее базовой математике (сложение и умножение).

## Описание
- Написано на Java для платформы Android.
- Генерирует случайные математические примеры разного уровня сложности.
- Предоставляет три варианта ответа с игровой системой начисления очков.
